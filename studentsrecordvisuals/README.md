## Project Description

This project reads student grade data from a CSV file and generates a simple grade analysis report.

The program analyzes student scores, calculates statistics, assigns grade levels, and generates a report file automatically.


## Features

- Read student data from `students.csv`
- Calculate basic statistics
- Classify scores into A/B/C/D/F grades
- Count grade distribution
- Generate `grade_report.txt`
- Create a grade distribution chart using matplotlib
